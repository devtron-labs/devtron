Deployment Templates

Operations for managing deployment templates including:

  • Application deployment template configuration
  • Template creation and updates
  • Pre/post deployment stage management
  • ConfigMap and Secret integration

Configure deployment template for application

Creates or configures a deployment template for an application including:

  • Deployment template YAML configuration
  • Pre-stage and post-stage configurations
  • ConfigMap and Secret references for stages
  • Environment execution settings
  • Cluster CD activation settings
Authorizations:
bearerAuth
Request Body schema: application/json
required

Application template configuration request

appId
integer <int64>

Application ID (required for updates)

deploymentTemplate
required
string

Deployment template as YAML string

object (StageConfig)

Pre/Post deployment stage configuration

object (StageConfig)

Pre/Post deployment stage configuration

object (ConfigMapSecretNames)

ConfigMap and Secret names for stage configuration

object (ConfigMapSecretNames)

ConfigMap and Secret names for stage configuration

runPreStageInEnv
boolean
Default: true

Whether to run pre-stage in environment

runPostStageInEnv
boolean
Default: false

Whether to run post-stage in environment

isClusterCdActive
boolean
Default: true

Whether cluster CD is active
