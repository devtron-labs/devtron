Devtron Support: support@devtron.ai URL: https://devtron.ai/support License: Apache 2.0 Terms of Service

This API provides functionality for managing applications in Devtron, including creating, updating, and listing applications, as well as managing application configurations, environments, and deployments.

List applications

Retrieves a paginated list of applications based on the provided filters. Can return deployed applications, unDeployed applications, or applications with incomplete configurations.

Authorizations:
bearerAuth
Request Body schema: application/json
required

Application listing filters

teamIds
Array of integers <int64> [ items <int64 > ]

Filter by team IDs

environmentIds
Array of integers <int64> [ items <int64 > ]

Filter by environment IDs

statuses
Array of strings
Items Enum: "Healthy" "Degraded" "Failed" "Progressing"

Filter by application statuses

appNameSearch
string

Search term for application name

offset
integer

Pagination offset

size
integer

Page size

projectIds
Array of integers <int64> [ items <int64 > ]

Filter by project IDs
