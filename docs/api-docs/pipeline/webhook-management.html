Webhook Configuration

Operations for managing webhook configurations

Create webhook configuration

Creates a new webhook configuration for event notifications

Authorizations:
bearerAuth
Request Body schema: application/json
required

Webhook configuration creation request

name
required
string [ 1 .. 100 ] characters

Webhook configuration name

description
string

Webhook description

webhookUrl
required
string <uri>

Webhook URL endpoint

httpMethod
string
Default: "POST"
Enum: "POST" "PUT" "PATCH"

HTTP method for webhook calls

Array of objects (WebhookHeader)

HTTP headers to include in webhook calls

payload
object

Webhook payload template

eventTypes
required
Array of strings non-empty
Items Enum: "DEPLOYMENT_SUCCESS" "DEPLOYMENT_FAILURE" "CI_SUCCESS" "CI_FAILURE" "APPLICATION_CREATED" "APPLICATION_DELETED"

Event types that trigger this webhook

object (WebhookFilters)
active
boolean
Default: true

Whether the webhook is active

object (RetryConfig)

Responses

Devtron Orchestrator API Server

/orchestrator/webhook/create
