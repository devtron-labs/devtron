Application Metrics

Operations for retrieving application metrics

Get application metrics

Retrieves metrics for applications including CPU, memory, network, and storage usage

Authorizations:
bearerAuth
Request Body schema: application/json
required

Metrics query request

appId
integer <int64>

Application ID

environmentId
integer <int64>

Environment ID

startTime
string <date-time>

Start time for metrics query

endTime
string <date-time>

End time for metrics query

step
string

Query step interval (e.g., "5m", "1h")

metricTypes
Array of strings
Items Enum: "cpu" "memory" "network" "storage" "replicas"

Types of metrics to retrieve

Responses

Devtron Orchestrator API Server

/orchestrator/metrics/application

Application Logs

Operations for retrieving application logs

Get application logs

Retrieves logs for applications with filtering and search capabilities

Authorizations:
bearerAuth
Request Body schema: application/json
required

Logs query request

appId
integer <int64>

Application ID

environmentId
integer <int64>

Environment ID

podName
string

Pod name (optional)

containerName
string

Container name (optional)

startTime
string <date-time>

Start time for log query

endTime
string <date-time>

End time for log query

logLevel
string
Enum: "DEBUG" "INFO" "WARN" "ERROR" "FATAL"

Minimum log level to retrieve

searchQuery
string

Search query for log filtering

limit
integer [ 1 .. 1000 ]

Maximum number of log entries to return

nextToken
string

Token for pagination
