Workflow Management API (1.0.0)

Download OpenAPI specification:

Devtron Support: support@devtron.ai URL: https://devtron.ai/support License: Apache 2.0 Terms of Service

This API provides functionality for managing workflows in Devtron, including creating, updating, and listing workflows, as well as managing workflow configurations, environments, and deployments.

Workflow Creation

CreateAppWorkflow

Creates a new workflow for the configurations provided.

Request Body schema: application/json
required

A JSON object containing the app configuration

integer

name
string

appId

cdPipelines
