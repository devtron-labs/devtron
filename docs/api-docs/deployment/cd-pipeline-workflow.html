Get CD Pipeline Workflow History

Retrieves the deployment history for a specific CD pipeline in an environment. Returns a list of workflow runs with their status, timestamps, and metadata.

Use Cases:

  • View deployment history for troubleshooting
  • Track deployment frequency and success rates
  • Audit deployment activities

Required Permissions:

  • Application view permission
  • Environment view permission
Authorizations:
bearerAuth
path Parameters
appId
required
integer >= 1
Example: 123

Unique identifier of the application

environmentId
required
integer >= 1
Example: 456

Unique identifier of the environment

pipelineId
required
integer >= 1
Example: 789

Unique identifier of the CD pipeline

query Parameters
offset
integer >= 0
Default: 0

Number of records to skip for pagination

size
integer [ 1 .. 100 ]
Default: 20
Example: size=20

Number of records to return per page

Responses

Get CD Pipeline Workflow Logs

Retrieves pre-deployment and post-deployment logs for a specific workflow run. Supports both streaming and historical log retrieval.

Use Cases:

  • Debug deployment failures
  • Monitor deployment progress
  • Audit deployment activities

Required Permissions:

  • Application view permission
  • Environment view permission
Authorizations:
bearerAuth
path Parameters
appId
required
integer >= 1
Example: 123

Unique identifier of the application

environmentId
required
integer >= 1
Example: 456

Unique identifier of the environment

pipelineId
required
integer >= 1
Example: 789

Unique identifier of the CD pipeline

workflowId
required
integer >= 1
Example: 12345

Unique identifier of the workflow run

query Parameters
logType
string
Default: "all"
Enum: "pre-deployment" "post-deployment" "all"
Example: logType=all

Type of logs to retrieve

follow
boolean
Default: false

Whether to stream logs in real-time
